# Contributing to Ramanans Birthday Bot

Contributions are welcome! Please follow these guidelines:

1. **Fork the Repository** to your own account.
2. **Create a Feature Branch** (`git checkout -b feature/mail-enhancement`).
3. **Commit Changes:** Make sure you don't commit credentials or personal `.env` files (these should be excluded via `.gitignore`).
4. **Open a Pull Request** describing your adjustments.

Thank you!
